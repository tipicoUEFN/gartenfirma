## Important Production Note

Current project setup uses temporary remote image URLs for speed. Before going live:

1. Download selected files locally.
2. Save under `public/images/...` with clear names.
3. Optimize to WebP/AVIF (target < 250 KB for hero).
4. Update `src/config/imageLibrary.js` to local paths.
5. Keep a record of source URL + license page + download date.
